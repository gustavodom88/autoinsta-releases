# AutoInsta — Releases

Repositório público de instaladores oficiais do AutoInsta.

## Onde baixar

Acesse [releases](https://github.com/gustavodom88/autoinsta-releases/releases/latest) e baixe:

- **Mac (Apple Silicon — M1/M2/M3):** `AutoInsta-X.X.X-arm64.dmg`
- **Mac (Intel):** `AutoInsta-X.X.X.dmg`
- **Windows:** `AutoInsta Setup X.X.X.exe`

## Como instalar

Veja o guia: [INSTALACAO.md](https://github.com/gustavodom88/autoinsta/blob/master/Sistema/docs/INSTALACAO.md)
